# DP-05 Walkthrough — Teenage Engineering Phisism Time System

## 完成した機能

### 上半分：時計 + カレンダー（同一パネル）
- **巨大時計** — `HH:MM:SS`、秒は82%スケール
- **コロン** — 2Hz点滅、opacity 100%↔20%（完全消灯禁止）
- **カレンダー** — 月グリッド、今日凹ハイライト、0.5pxグリッド線

### 下半分：4モジュール横並び

| DAY | REMAIN | NET | DEVICES |
|-----|--------|-----|---------|
| 進行バー + 目盛り + % | 残時間 HH:MM:SS | DL/UL Mbps + スパークライン | ●/○ 状態リスト |

### 設定パネル
- 下半分長押し → 各モジュールが設定項目にインプレース置換（カレンダー・時計は消えない）
- フォントアップロード / ウェイト / 秒スケール / Thin Override / Monospace / Night Mode / Glow Color / 焼き付き防止

### Night Vision モード
- 背景 `#0B0B0C`、選択発光色（赤/緑/アンバー）
- 影 → 発光縁取り、秒更新時3%グロー増幅

### 実データ連携
- **Vite サーバーブリッジ**: `systeminformation` を使用した Node.js 側 API を実装。
- **NET**: 実際のネットワークインターフェースの送受信量を 1s 毎に取得し、Mbps 表示とスパークラインに反映。
- **DEVICES**: MacBookのホスト名 (`.local`) および実際に接続されているUSB/ネットワークデバイスをリスト表示。

### その他
- 微細ノイズ（1.8%、SVG filter）
- 環境光グラデーション（135°、2%明度差）
- 焼き付き防止（Sub-Pixel Drift Matrix、60s/120sシフト）
- フォント疑似モノスペース化（em単位グリフ幅計測）
